
# Tourism Experience Analytics  
**Classification • Prediction • Recommendation System**

## Overview

This project analyzes real-world tourism data to:
- Predict user ratings for attractions (Regression)
- Classify visit mode (Classification)
- Provide personalized attraction recommendations (Collaborative Filtering)
- Deliver interactive insights via Streamlit dashboard

## Key Results

- Ratings: 80%+ are 4 or 5 (strong positive bias)
- Dominant modes: Family & Couples (60%+ visits)
- Attractions: Only 30 unique items → high concentration
- Seasonality: Peak in June–August
- Loyalty: ~80% one-time visitors

## Tech Stack

- Python 3.9+
- Pandas, NumPy
- Scikit-learn (Random Forest, TF-IDF, etc.)
- Matplotlib, Seaborn
- Streamlit (dashboard)
- NLTK (text preprocessing)
